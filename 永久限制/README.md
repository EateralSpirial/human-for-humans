# 永久限制

本目录用于记录经人工确认的申诉流程滥用行为。

## 适用情形

提交者存在以下行为时，维护者可以将其加入永久限制记录：

1. 伪造申诉信息。
2. 批量提交无事实基础的申诉。
3. 使用申诉流程骚扰维护者或其他账号主体。
4. 多次提交攻击性、侮辱性或无关内容。
5. 其他明显滥用申诉流程的行为。

## 处理结果

进入永久限制记录的提交者，其后续申诉可以被直接关闭，除非维护者主动决定复核。

## 记录方式

记录文件：`申诉滥用记录.json`

每条记录建议包含：

```json
{
  "GitHub用户": "",
  "相关Issue": [],
  "原因": "",
  "处理日期": "",
  "处理者": "",
  "备注": ""
}
```
